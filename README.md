## Tuvok
The logical Jekyll theme for an elegant GitHub Pages blog.

![tuvok-home-screenshot](assets/images/tuvok-home-screenshot.png)

## Installation

  git clone https://github.com/danrpts/tuvok.git
  cd tuvok
  bundle install

## Usage

  bundle exec jekyll serve
